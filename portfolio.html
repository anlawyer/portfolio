Gotta Go!

Gotta Go is a crowdsourced application made for people who are always on the road. The app provides users with location and information about nearby bathrooms that other users have input.


My team and I used Node.js, Express.js, MySQL and the Sequelize ORM for the backend,and Bulma and EJS for the frontend.

Eat Da Burger!

Eat Da Burger is a full-stack application using Node.js, Express.js, MySQL and Flexbox. Users can add a burger to the database using the form provided, then click a button to "devour" the burger, which is then moved from the uneaten list to the eaten list.

Friend Finder

Friend Finder is a full-stack application using Node.js, Express.js and Bootstrap. Users take a survey and are matched to a user who has also taken the survey through code that finds the least amount of difference between their answers.


Live app | Source code

Bamazon

Bamazon is a backend command line application using Node.js, MySQL and the Inquirer NPM package. It functions as a store-front, showing customers available products and allowing them to "purchase" and deplete the stock. It also allows managers to view all products and inventory, and update the stock. The database is updated using CRUD operations.

GifTastic

GifTastic is a front-end application that uses the Giphy API to display gifs on the click of a button. The user is able to add buttons of any topic, and click on each gif to start it playing and stop it playing.
